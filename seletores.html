Seletores CSS

Por tag

Utilizar o nome da própria tag para estilizar no CSS:

p {
     /* seu estilo vai aqui */
}

Utilizar com precaução, pois podemos ter vários parágrafos, e querermos estilos diferentes.

=> Temos também o seletor universal (*), que seleciona todos os elementos da página.

Seletores por ID e por classe

A tag no HTML deve ter o atributo id e/ou class e seu valor:

<h1 id="tituloPrincipal">

<p class="texto">

Para chamar um elemento pelo id no CSS, utilizamos # (hashtag). Para chamar uma classe, utilizamos . (ponto).

#tituloPrincipal {
     /* seu estilo vai aqui */
}

.texto {
     /* seu estilo vai aqui */
}
