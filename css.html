Propriedades CSS

Fontes e texto

=> color: altera a cor da fonte, podemos passar um valor em rgb, rgba, hexadecimal, ou o nome da cor

=> font-size: altera o tamanho da fonte

=> font-weight: altera a expessura da fonte

=> font-style: altera o estilo da fonte, podendo ser itálico, sublinhado (underline)

=> text-align: alinha o texto

Posição e espaçamentos

=> position: absolute - permite posicionar um elemento passando valores para o top, right, bottom e left. O elemento vai estar com posição absoluta em relação a outro, que recebe a propriedade e valor position: relative

=> margin: espaçamento externo, valor auto para centralizar horizontalmente

=> padding: espaçamento interno (com box-sizing: border-box esse valor é tirado do total da width e do height)
