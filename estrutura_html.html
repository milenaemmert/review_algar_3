Estrutura HTML

DOCTYPE, html, head e body

=> Doctype informará ao navegador qual tipo de documento ele deve esperar, no caso, do tipo html.

=> <html></html> engloba as tags <head></head> e <body></body>

=> Dentro das tags <head></head> colocamos informações para o navegador, como o <title></title> (título da aba), links para CSS, meta tags.

=> E no <body></body> colocamos todo o conteúdo do site.

Tags básicas

=> <h1></h1> - títulos por ordem de importância (do nível 1 até o 6, sendo o 1 o mais importante)

=> <a></a> - navegação entre páginas web; indicar no atributo href a URL para qual o hiperlink aponta; target="_blank" abre o link em nova guia.

<p></p> - representa um parágrafo

<div></div> - tag genérica para agrupar um conteúdo.

Tags semânticas

Ajuda mecanismos de buscas e facilita a acessibilidade das páginas web devido as suas tags de usos bem definidos.

=> <header></header> - tag de cabeçalho

=> <main></main> - conteúdo mais relevante

=> <section></section> - agrupa conteúdos como uma div, mas é uma tag semântica

=> <footer></footer> - rodapé

Tags semânticas

=> <nav></nav> - menu de navegação

=> <ul></ul> - agrupa uma lista não ordenada

=> <ol></ol> - agrupa uma lista ordenada

=> <li></li> - item da lista, seja ela ordenada ou não
